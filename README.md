# File Organizer Automation

A beginner-friendly Python automation script that organizes files in a folder
based on their file type (Documents, Images, Videos, etc.).

## Features
- Automatically sorts files into folders
- Supports multiple file extensions
- Prevents overwriting existing files

## Tech Used
- Python
- OS module
- shutil

## How to Run
1. Clone the repository
2. Place the script in the folder you want to organize
3. Run:
   python file_organizer.py
